# React.Net-Router-Redux

A simple isomorphic boilerplate SPA example using ReactJs.Net, React-Router and Redux.

# Libraries

 * React 0.14.x
 * React Router
 * Redux
 * Webpack
 * Babel
 * React Bootstrap

# Usage

* Update all Nuget packages
```
Update-Package
```

* Install NPM packages
```
cd App && npm install
```

* Build the React App
```
webpack
```

Now you can build and run the solution.
